# コスト試算書

## 月額コスト目標: 1,000円以内

## 1. AI分析コスト（Vertex AI - Gemini 3 Flash）

### 想定使用量

| 項目 | 数値 |
|------|------|
| ニュース収集頻度 | 2回/日 |
| 1回あたりの記事数 | 平均1件 |
| 月間記事数 | 60件 (2回/日 × 30日) |
| 1記事あたりの入力トークン | 1,500トークン |
| 1記事あたりの出力トークン | 500トークン |

### 料金計算

**⚠️ 重要: Gemini 3 Flashへの料金改定**
- Gemini 1.5 Flash → Gemini 3 Flashで料金が大幅に改定されました
- 入力トークン: 約13倍の値上げ
- 出力トークン: 約20倍の値上げ

**Gemini 3 Flash 料金 (2024年12月〜)**:
- 入力: $0.50 / 1M トークン (旧: $0.0375)
- 出力: $3.00 / 1M トークン (旧: $0.15)

### Grounding with Google Search使用

**想定**: 1日2回、各回3件のニュース検索

**月間コスト**:
```
1. Gemini API利用料
   入力: 60回 × 2,000トークン × $0.50/1M = $0.06
   出力: 60回 × 1,000トークン × $3.00/1M = $0.18
   小計: $0.24

2. Grounding料金 (2026年1月5日〜)
   - 1,500検索/日まで無料
   - 60回/月 × 3検索 = 180検索/月 → 無料枠内
   小計: $0

合計: $0.24 ≈ 約36円
```

**年間コスト**: 約432円

## 2. Firestore コスト

### 想定使用量

| 項目 | 数値 |
|------|------|
| ニュース保存数 | 60件/月 |
| 取引履歴保存数 | 10件/月 (想定) |
| ポジション更新頻度 | 100回/月 (想定) |
| ダッシュボード読み取り | 60回/月 (1日2回) |

### 料金計算

**Firestore 料金** (asia-northeast1):
- ドキュメント書き込み: $0.18 / 100,000回
- ドキュメント読み取り: $0.06 / 100,000回
- ストレージ: $0.18 / GB

**月間コスト**:
```
書き込み: (60 + 10 + 100) × $0.18/100,000 = $0.000306
読み取り: 60 × $0.06/100,000 = $0.000036
ストレージ: 0.01GB × $0.18 = $0.0018
合計: $0.00214 ≈ 約0.3円
```

### データ削除による節約

3ヶ月以上前のデータを削除することで、ストレージコストを最小化:
- 最大保持データ: 約180件 (60件/月 × 3ヶ月)
- ストレージサイズ: 約0.03GB以下

**年間コスト**: 約4円

| CPU使用時間 | 10秒/リクエスト |

### 料金計算

**Cloud Run 料金** (asia-northeast1):
- CPU: $0.00002400 / vCPU秒
- メモリ: $0.00000250 / GiB秒
- リクエスト: $0.40 / 100万リクエスト

**月間コスト**:
```
CPU: 60リクエスト × 10秒 × 1vCPU × $0.000024 = $0.0144
メモリ: 60リクエスト × 10秒 × 0.5GiB × $0.0000025 = $0.00075
リクエスト: 60 × $0.40/1,000,000 = $0.000024
合計: $0.01518 ≈ 約2円
```

### 無料枠

Cloud Runは無料枠あり:
- CPU: 180,000 vCPU秒/月
- メモリ: 360,000 GiB秒/月
- リクエスト: 200万リクエスト/月

→ **本システムは無料枠内に収まるため、実質0円**

**年間コスト**: 約0円 (無料枠内)

## コスト増加リスクと対策

### リスク1: ニュース収集頻度の増加

**想定**: 1日2回 → 1日6回に増加

**影響**:
- Vertex AI (Grounding): 約36円 → 約108円 (+72円)
- Cloud Functions: 無料枠内 → 無料枠内 (影響なし)

**対策**: 頻度増加しても予算内に余裕で収まる

---

### リスク2: 取引頻度の増加

**想定**: 月10回 → 月100回に増加

**影響**:
- Firestore書き込み: 約0.3円 → 約1円 (+0.7円)

**対策**: 取引頻度が高くてもコストインパクトは小さい

---

### リスク3: X API自動投稿の開始

**想定**: Basic プラン契約 ($100/月)

**影響**: 月額2円 → 約15,000円

**対策**:
- 投稿頻度を週1回に制限 (月4回まで)
- 手動投稿を継続
- X API Freeプランの復活を待つ

---

### リスク4: ダッシュボードアクセス頻度の増加

## OpenAI API との比較

### GPT-4o-mini を使用した場合

**OpenAI API (GPT-4o-mini) 料金**:
- 入力: $0.15 / 1M トークン
- 出力: $0.60 / 1M トークン

**月間コスト**:
```
入力: 60記事 × 1,500トークン × $0.15/1M = $0.0135
出力: 60記事 × 500トークン × $0.60/1M = $0.018
合計: $0.0315 ≈ 約5円
```

| 項目 | Gemini 3 Flash (Grounding) | OpenAI GPT-4o-mini |
|------|---------------------------|-------------------|
| AI分析コスト | 約36円/月 | 約5円/月 |
| 認証管理 | GCP IAM (無料) | API Key管理 |

しかし、Grounding機能による最新ニュース取得という独自の価値があり、
予算1,000円/月に対して4%程度なので許容範囲内です。

### コスト配分（Grounding使用時）

```
Vertex AI (Gemini): 97% (約36円)
Firestore:          1%  (約0.3円)
BigQuery:          0.3% (約0.1円)
```

### 年間コスト: 約437円

約437円/年で運用可能です。

40年運用想定でも**約1.7万円**と非常に低コストで運用できます。

### 余裕度

月額予算1,000円に対して、実際のコストは約37円のため、**96%の余裕**があります。

この余裕を活用して:
- ニュース収集頻度を増やす（1日6回でも約108円）
- より高度な分析機能を追加
- バックテスト実行回数を増やす
- 複数の通貨ペアに対応

など、大幅な機能拡張が可能です。

### ⚠️ 料金改定の影響

Gemini 1.5 Flash（旧料金: 約1.2円/月）からGemini 3 Flash + Grounding（新料金: 約36円/月）への移行により、
AI分析コストは**約30倍**に増加しましたが、予算の4%程度であり十分許容範囲内です。
